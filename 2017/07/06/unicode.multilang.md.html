I like the – for spans (lived 1850–1920).[Wikipedia about Dash](https://en.wikipedia.org/wiki/Dash) ## How to type accents ? - On Mac & Linux if you have a keyboard layout that can type accent but no capital letters with accents: Caps Lock then é then Caps Lock gives you a **É**, the key is a Caps Lock (put all letters in Capital Letter) and not a Shift Lock (act as if Shift was pushed). The latter behaviour is due to the typewritters when the key induced a physical move. On Windows, the Shift Lock behaviour is there. Linux behaviour can of course be changed. - On Windows (Microsoft Office only) support for accents via Ctrl+' then E, cedilla is ,, Trema/Umlaut is ". ## How to type any Unicode character, including accents? Best methods at the **TOP** of the list: - On Mac & Linux, your keyboard have probably more symbols (AltGr and AltGrShift are filled with a lot of characters on my keyboard, like AltGr+I = **→**) or use extended layout with the symbols you want or create yours. - On Linux Use Compose key !!! It does not depend on keyboard layout at all, I've mapped it to Pause because no app uses that. Go in system settings to choose where to map it. Then for example type Compose then ' then e gives **é** or Compose < < gives **«** or Compose > = gives **≥** and a lot more on [Ubuntu Help](https://help.ubuntu.com/community/ComposeKey). - On Android/iOS : long press on a key to get more symbols - On Mac (Cocoa App only) long press on a key will pop up - On Latex, there is probably a sequence like `\times` for **×** - On Html, there is probably a sequence like `×` for **×** [Wikipedia List](https://en.wikipedia.org/wiki/List_of_XML_and_HTML_character_entity_references) - On vim: `digraph`!! That's the same idea as the Compose Key. - On Any platform: Google the name and copy paste... or go in the OS Character Map app. But use this as a last resort because it is so slow. ### If you know the Unicode codepoint like U+2205 - On Linux GTK+ App: Ctrl+Shift+U 2 2 0 5 Enter. - On Windows Alt and + on keypad (requires registery setting [Method 1](http://www.fileformat.info/tip/microsoft/enter_unicode.htm) [Method 2](https://www.johndcook.com/blog/2008/08/17/three-ways-to-enter-unicode-characters-in-windows/)) then 2 2 0 5. But prefer other input method it's stupid to memorise the numbers... - On Html: `∅` ### If you know how to write it by hand - For LaTeX: [detexify](http://detexify.kirelabs.org/classify.html), then [LaTeX to unicode](http://vikhyat.net/projects/LaTeX_to_unicode/) might work. - For Unicode there is (but that's harder): [shapecatcher](http://shapecatcher.com/) ### Notes on French keyboards - On French from France Windows keyboard, no way to input Accent directly from keyboard, change to Belgian or use other methods in that list. - On Belgian Keyboard there is a dead key for Acute accent **and** Grave accent on AltGr+ù and AltGr+µ. - None of those keyboards gives you **Ç**. {% endmarkdown %}
