Python also have `import unicodedata` module for more advanced Unicode-related operation. - Numeric ↔ Character, `chr(233) == chr(0xe9) == "é"`, `ord("é") == 233` - Uppercase, lowercase, `"é".upper() == "É"`, `"é".lower() == "é"` - Name query, `unicodedata.name("é") == 'LATIN SMALL LETTER E WITH ACUTE'` - Composition, decomposition, `unicodedata.normalize('NFC', "é") # U+00E9`, `unicodedata.normalize('NFD', "é") # U+0065 U+0301` Here are my recipes for fast operations in calculator style: import unicodedata def uniname(s): """ é → LATIN SMALL LETTER E WITH ACUTE """ if len(s) == 1: return unicodedata.name(s) else: return [uniname(x) for x in s] def uord(s): """ é → U+00E9 """ if len(s) == 1: return 'U+' + hex(ord(s))[2:].zfill(4).upper() else: return [uord(x) for x in s] def uniline(s): """ é → LATIN SMALL LETTER E WITH ACUTE é U+00E9 """ if len(s) == 1: return uord(s) + ' ' + s + ' ' + uniname(s) else: return [uniline(x) for x in s] You can then use in the interactive shell `uniline("αβγ")` to have a nice list: ["U+03B1 α GREEK SMALL LETTER ALPHA", "U+03B2 β GREEK SMALL LETTER BETA", "U+03B3 γ GREEK SMALL LETTER GAMMA"] {% endmarkdown %}
